# Wave Modes

**Motive** and **Corrective**

**Motive**

![Docusaurus logo](/img/5-wave-pattern.png)

- a 5 wave structure


**Corrective**

![Docusaurus logo](/img/abc-simple.png)

- a 3 wave structure
- counter trend interruptions (include wave 2 and wave 4)
- appears as a response to the preceding motive wave
